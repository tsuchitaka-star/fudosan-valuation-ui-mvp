# 不動産価値診断 UI MVP

不動産価値診断サービスの **フロントエンドUI改善MVP** です。  
建設会社サイトのようなトーンで、トップ〜物件種別〜入力〜結果までの流れを確認できます。

> これは UI / UX のデモです。本物の査定API・DB・ログイン機能は未実装です。

## 何ができるか

- トップページ（ヒーロー + スクロール演出）
- 物件種別選択（マンション / 土地 / 土地+建物）
- 必須入力（住所・駅・面積など）
- 任意詳細入力
- 結果画面（価格帯・補正・デモ近隣事例）

## まだないもの

- バックエンドAPI
- 実勢価格データ連携
- 認証 / 会員機能
- 本番用の査定エンジン

結果の金額は `app.js` 内のデモ計算（`mockValuation`）です。見た目と導線確認用です。

## 起動方法

### いちばん簡単

`index.html` をブラウザで直接開く。

### ローカルサーバ（推奨）

Python がある場合:

```bash
# このフォルダで
python -m http.server 4173
```

その後ブラウザで開く:

```text
http://localhost:4173/
```

Node.js がある場合:

```bash
npx serve .
```

## 触ってほしい流れ

1. トップで「無料で診断を始める」
2. 物件種別を選ぶ
3. 住所・駅・面積を入れる
4. 詳細を入れる or スキップ
5. 結果画面で価格帯を確認

## 友達向けメモ

- 見た目と体験確認用のフロントのみ
- 金額は仮計算なので、本番ロジックとして使わないでください
- フィードバック歓迎ポイント:
  - トップの第一印象
  - 入力の分かりやすさ
  - 結果画面の信頼感
  - スマホ見た目

## 技術メモ

- 静的HTML / CSS / 素のJavaScript
- フレームワークなし
- 外部API呼び出しなし
- 画像は `assets/` 配下
